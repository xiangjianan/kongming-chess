# 🎮 孔明单人棋

经典的单人跳棋益智游戏，又称"独立钻石"或"孔明棋"。

## 🎯 在线游玩

**👉 [https://xiangjianan.github.io/kongming-chess/](https://xiangjianan.github.io/kongming-chess/)**

## 🎲 游戏规则

1. 棋子只能跳过相邻的棋子到达空位
2. 被跳过的棋子会被吃掉
3. 目标是让棋盘上只剩下一个棋子

## 🕹️ 操作方式

- 点击棋子选中
- 绿色高亮显示可移动位置
- 点击目标位置完成跳跃

## ✨ 功能特性

- 🎯 10 个精心设计的关卡
- ↩️ 撤销/重置功能
- 📊 步数和剩余棋子统计
- 🎨 木质风格界面设计

## 📱 技术栈

- 纯 HTML + CSS + JavaScript
- 无需后端，纯前端实现
- 响应式设计，支持移动端
